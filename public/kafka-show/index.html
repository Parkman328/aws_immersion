Display Kafka Messages

Here are the 200 most recent messages that were delivered to Kafka displayed using the Kafka “Console Consumer”. Messages are deserialized as raw text, but as you can see the data is in JSON format as we configured in the Replicate task.

     /* Add redirection so we can get stderr. */
     $cmd="/opt/kafka*/bin/kafka-console-consumer.sh --bootstrap-server kafka:29092  --topic testdrive --offset ${offset} --max-messages ${msgcount} --partition 0 2>&1";
     $handle = popen("$kafkaopts;$cmd", 'r');

     while (!feof($handle)) {
        $output = fread($handle, 2096);
        echo(nl2br($output));
        ob_flush();
        flush();
     }
     pclose($handle);
