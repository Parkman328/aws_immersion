Log In to Qlik Replicate to Start the Tutorial

First things first, we need to open Qlik Replicate. Open Replicate in a new tab in your broswer. For host username and password please refer to GettingStarted Page

Please note that you may need to press accept security and proceed for certificate

Once you are logged in you will see the main screen for Qlik Replicate.
